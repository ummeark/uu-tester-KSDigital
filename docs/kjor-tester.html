ℹ️ Slik fungerer det
  1. Klikk en testknapp over – du sendes til GitHub Actions i ny fane
  2. Logg inn på GitHub hvis du ikke allerede er innlogget
  3. Klikk «Run workflow» (grønn knapp øverst til høyre), velg riktig test i nedtrekkslisten
  4. Gå tilbake til denne siden – status oppdateres automatisk hvert 15. sekund under kjøring
  5. Når testen er ferdig: resultater er automatisk publisert til GitHub Pages ✅

Forutsetning: Mac-en må være på og self-hosted runner må kjøre. Oppsett: Repo → Settings → Actions → Runners → New self-hosted runner.
